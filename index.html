Jenis Transportasi Umum

Bus

Bus adalah salah satu moda transportasi yang paling umum dan mudah diakses.

Kereta Api

Kereta api adalah pilihan yang baik untuk perjalanan jarak jauh.

Metro

adalah transportasi cepat untuk area perkotaan.

Taxi

adalah layanan antar jemput menggunakan mobil

Gojek

merupakan layanan transportasi online dengan kendaraan roda dua atau motor

LRT

merupakan transportasi yang sama dengan kereta api

MRT

merupakan transportasi yang sama dengan kereta api dan juga LRT

Angkot

merupakan transportasi umum yang yang menggunakan mobil untuk menjadi kendaraan nya

Tiket dan Tarif

Informasi tentang harga tiket.

Tiket Bus

Harga Tiket Bus bervariasi mulai dari Rp.50.000 - Rp.250.000

Tiket Kereta Api

Harga Tiket Kereta Api bervariasi tergantung kita berangkat dari stasiun asal, harga nya Rp.3.000 untuk 25 km pertama dan Rp.1.000 untuk setiap 10 km berikutnya

Tiket Metro

Harga Tiket Metro bervariasi mulai dari Rp. 15.000 - Rp.50.000

Tiket Taxi

Harga Tiket Taxi kurang dari 10 kilometer akan dikenakan Rp6.000 per kilometer. Kemudian untuk jarak 10 sampai 30 kilometer akan dikenakan tarif Rp5.200 per kilometer. Dan, perjalanan lebih dari 30 kilometer akan dikenakan tarif Rp5.000 per kilometer

Tiket Gojek

Harga Tiket Khusus wilayah JABODETABEKbiaya GoRide menjadi Rp3.000/km untuk perjalanan di atas 10km.

Tiket LRT

Harga Tiket LRT Rp 5.000 untuk satu kilometer pertama. Selanjutnya untuk tarif maksimal adalah Rp 10.000 hingga Rp 20.000 sesuai jam sibuk.

Tiket MRT

Harga Tiket MRT single trip dengan harga Rp15.000 dan multi trip dengan harga 25.000.

Tiket Angkot

Harga Tiket Angkot bervariasi mulai dari Rp.5.000 - Rp.15.000
